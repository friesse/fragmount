CS:GO UPDATES

March 15, 2025
Release Notes for 3/15/2025

Today's update includes several improvements to gameplay and fixes for reported issues:

  • Fixed a bug that caused incorrect player animations when switching weapons
  • Improved server performance on high player count matches
  • Adjusted the smoke grenade effect to better match its collision shape
  • Fixed a rare crash that could occur when encountering specific network conditions
  • Updated weapon balance for the SSG-08 to reduce movement accuracy
March 8, 2025
Release Notes for 3/8/2025

We've deployed a small update with the following changes:

  • Updated de_ancient to fix several boost spots
  • Added a new sound effect for headshot feedback
  • Fixed a visual glitch with certain player models on low shader settings
  • Fixed an issue with team logo displays in spectator mode
February 25, 2025
Major Weapon Balance Update

After analyzing play data from recent tournaments and community feedback, we've made several balance changes:

  • Adjusted AK-47 recoil pattern to be slightly more predictable
  • Reduced M4A1-S magazine size from 25 to 20
  • Increased Deagle accuracy on first shot
  • Reduced P90 price by $50
  • AWP movement penalty increased by 5%
  • MP5-SD damage increased at medium range

These changes aim to promote more diverse weapon selection in competitive play while maintaining the core gunplay that makes CS:GO unique.

February 12, 2025
Map Updates and Server Improvements

This update focuses on map optimization and server infrastructure:

  • Updated de_dust2 with improved lighting and texture optimization
  • Added a new callout position in de_inferno "Dark Kitchen"
  • Modified bomb site B on de_nuke to include an additional entry point
  • Deployed server-side optimization for improved netcode performance
  • Reduced server tick variance during high-load scenarios

The map changes should provide enhanced visibility and more tactical options for both sides.
